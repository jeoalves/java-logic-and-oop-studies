# Java Fundamentals & OOP Studies ☕

This repository contains my personal studies and exercises developed during my journey to becoming a Java Backend Developer. It covers everything from basic logic to Advanced Object-Oriented Programming (OOP) concepts.

## 🚀 Key Topics Covered
- **Basic Logic:** Variables, types, and control flow (`if`, `while`, `for`).
- **Object-Oriented Programming:** Classes, Objects, Attributes, and Methods.
- **Encapsulation:** Practice with `Getters`, `Setters`, and Access Modifiers.
- **Advanced OOP:** Inheritance (`Super`), Polymorphism, and Interfaces.
- **Modern Java:** Lambda Expressions, Generics, and Serialization.

## 🛠️ Technologies Used
- **Java 17** (LTS)
- **Git & GitHub** for version control
- **IntelliJ IDEA / Eclipse** (IDE)

## 📖 Current Learning Path
I am currently following a deep-dive course into **Spring Boot** and **Cloud Architectures**. My goal is to master the Java ecosystem and transition into a **DevOps/Backend** international career.
